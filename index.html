Guess the Element!

▶️ Game Overview

This is a two-player game based on the first 20 elements of the periodic table. If you have multiple participants, the game may be played with two groups.

💡 How to Play

  • Players take turns guessing either the correct chemical symbol for a given element name, or the correct element name for a given symbol.
  • For each round, one of the 20 element-symbol pairs is randomly selected.
  • The direction of the question (name-to-symbol or symbol-to-name) is also chosen at random.

🔢 Scoring and Steals

  • A correct answer earns the player 1 point.
  • If the answer is incorrect, the other player has a chance to steal. A successful steal also earns 1 point and gives that player the next regular turn.
  • If the steal is incorrect or goes unanswered, the game simply moves on to a new question.
  • The game ends once all 20 element-symbol pairs have been used. The total combined score will always be exactly 20 points since each pair is only used once and one point is awarded per pair.

⏲️ Timing, Saving, and Resetting

  • Each turn — including steal attempts — is timed at 20 seconds. If no answer is submitted in time, the game proceeds to the next item.
  • Progress is automatically saved locally. If the page is refreshed or reopened mid-game, the game resumes from the last turn or steal, though the timer will start at 10 seconds instead of 20.
  • A Reset button is available to restart the game at any time after all 20 items have been answered.
  • A win page appears once all 20 items have been answered.
